This project uses historical stock data to:
Calculate expected returns, beta, covariance, and portfolio volatility
Simulate the efficient frontier through Monte Carlo simulations
Identify the Maximum Sharpe Ratio and Minimum Variance portfolios
Visualize portfolio performance and risk metrics

Key Features
Fetches stock Data using yfinance
Computes daily and annualized returns, volatility, and Sharpe ratio
Simulates thousands of random portfolios to plot the efficient frontier
Highlights optimal portfolios (Max Sharpe & Min Variance
Allows user-defined tickers (up to 3 for demonstration)

Technologies Used
Python
NumPy, Pandas — data handling & calculations
Matplotlib, Seaborn — data visualization
yfinance — fetching historical data
